Setup the development enviroment

Creating native apps is NOT recommented! Create OSSL apps! If it still doesn't work using OSSL, comment in the official OSSL project here! Native apps are helpful if you need to access features that are not implemented yet or require specific things that may be unsecure(When modding OnakOS it may be helpful)

To setup the development enviroment you firstly need to create a copy of OnakOS or you need to remix it

When copying you are not allowed to publish it however you can publish the app itself without the OS! Remixing and then publishing makes it for users easier to test your app without importing it somewhere

Now we finally can begin! Click Click Inside on the project page. You need to look for a sprite called apps//template. In that sprite there are all scripts that are required for an App.

You need to replace "example" with you app name(It should be like this Authorname.Appname)

The second required change is to replace in the broadcast "example" with your app nameAuthorname.Appname The other scripts should NOT be edited except you want to customize something

Change Loading Icon or something else! Do whatever you want as long as it is efficient and it works!
